Did you know that in JavaScript, certain values are passed by reference but others are passed by their values? There are two kinds of types: primitive types and reference types. Guess which ones are passed by reference?

Primitive types are:

  • Booleans
  • Strings
  • Numbers
  • Null
  • Undefined
Reference types are:
  • Arrays
  • Objects
  • Anything that's not a primitive type
