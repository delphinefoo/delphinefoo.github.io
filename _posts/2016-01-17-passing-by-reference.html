Did you know that in JavaScript, certain values are passed by reference but others are passed by their values? There are two kinds of types: primitive types and reference types. Guess which ones are passed by reference?

Primitive types are:

  1. Booleans
  2. Strings
  3. Numbers
  4. Null
  5. Undefined
Reference types are:

  1. Arrays
  2. Objects
  3. Anything that's not a primitive type

This weirdness happens because objects are passed by reference, not by value. This means that when we made y equal to x, we made y point to the same place in memory that x was pointing to, and they are pointing to the same object.

When you alter y, you are also altering x. This is important to remember when working with objects!
